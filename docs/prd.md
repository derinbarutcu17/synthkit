# PRD

## Codename

SynthKit

## Mission

Turn messy working material into structured briefs, decision memos, and deck outlines with citations, contradictions, confidence reporting, and revision history.

## v1 wedge

Build one sharp flow first:

- ingest messy material
- synthesize a brief or memo
- preserve citations and contradictions
- expose the result through MCP, CLI, HTTP, and TS SDK

## Out of scope for v1

- collaboration
- auth
- cloud hosting
- fancy web UI
- broader agent autonomy
