# Schema Contracts

The following schemas are versioned and stable at v1:

- `ProjectV1`
- `SourceV1`
- `SourceAssetV1`
- `ChunkV1`
- `SynthesisRequestV1`
- `ThemeClusterV1`
- `ContradictionV1`
- `DraftV1`
- `DraftSectionV1`
- `CitationV1`
- `ConfidenceReportV1`
- `RevisionV1`
- `ExportArtifactV1`
- `CapabilityManifestV1`

Every surface should map cleanly to these schemas.
